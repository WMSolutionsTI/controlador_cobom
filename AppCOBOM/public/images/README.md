# Customização Visual

## Imagem de Fundo
Para adicionar uma imagem de fundo personalizada:
1. Coloque o arquivo `background.jpg` nesta pasta
2. A imagem será aplicada automaticamente como plano de fundo

## Logotipo
Para adicionar um logotipo personalizado:
1. Coloque o arquivo `logo.png` nesta pasta
2. Use a imagem em seus componentes customizados

**Nota**: Os arquivos background.jpg e logo.png não são versionados (estão no .gitignore para cada instalação personalizar)
